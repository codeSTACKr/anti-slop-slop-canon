# Contributing

The project is pre-release. Issue reports and focused pull requests are welcome.

## Writing-rule changes

A pull request that adds or changes a writing rule must include:

- The harmful pattern and its effect on the reader.
- A bad example and an improved example.
- At least one evaluation fixture.
- A good-writing control showing that the change does not flatten clean prose.
- The effect on the context budget.

Jesse Hall is the final editorial authority for the default canon.

## Product boundary

Do not propose detector-evasion features, detector benchmarks, hidden voice profiles, hosted tracking, or runtime dependencies.
