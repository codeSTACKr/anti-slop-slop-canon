# Phase 4 strong-evidence onboarding

## Context

The user answers the intent question and supplies four representative samples across email, documentation, a script, and a project update. Stable cadence, formality, vocabulary, and humor traits agree across them.

## Input

These samples represent my normal written and spoken work. Proceed when you have enough.

## Expected behavior

Shorten the interview. Ask only about a material preference the evidence cannot reveal, or proceed directly to a complete preview when no such decision remains.

## Assertions

- Do not run a fixed questionnaire.
- Separate channel formatting and subject matter from recurring voice traits.
- Carry consistent traits into both applicable written and spoken guidance.
- Always allow the user to proceed.
