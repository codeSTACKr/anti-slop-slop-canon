# Installation

The repository is a pre-release implementation and is not ready to install. The planned installer is `npx skills add codeSTACKr/anti-slop-slop-canon -g`; omitting `-g` selects project scope.

No runtime package, script, API key, hosted inference service, or telemetry is part of the product. The Markdown runtime now includes routing, adaptive onboarding, and realtime style modules. Do not publish or recommend the planned command until profile lifecycle and release evaluation gates pass.
